Dependencies:

 -Express. HTTP, Routes, etc
 -BCrypt-NodeJS. Hash passwords
 -Body-Parser. Middleware to parse body requests into Javascript Object.
 -Connect-Multiparty. Allows to upload files
 -Mongoose. MongoDB object modeling tool designed to work in an asynchronous environment.
 -Mongoose-Pagination.
 -JWT-Simple. Tokens administration and authentication.
 -Moment. Work with dates and timestamps.
 -Nodemon. Each time code is changed in the node server, it restarts the server with the new changes.
